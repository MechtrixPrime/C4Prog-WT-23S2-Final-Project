# Portfolio Website Project

## Overview

This project is a personal portfolio website developed as part of the Certificate IV in Information Technology (Programming) Web Technologies unit.

The purpose of the website is to showcase current and future programming projects, skills, and professional information. The site is being developed using HTML, CSS, JavaScript, and TailwindCSS, with version control managed through Git and GitHub.

## Project Objectives

* Create a responsive portfolio website.
* Implement a mobile-first design approach.
* Use semantic HTML elements throughout the site.
* Integrate TailwindCSS into the development workflow.
* Showcase completed and future programming projects.
* Demonstrate web development skills and best practices.
* Maintain source code using Git version control.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* TailwindCSS
* Git
* GitHub
